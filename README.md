# Room Cost Calculator

A simple Python project that calculates:

- Room area
- Total painting/flooring cost

## Technologies
- Python

## Concepts Used
- Variables
- Input
- float()
- Arithmetic Operators
- print()


## Example

Input:
Length: 5.3
Width: 7.8
Price per square meter: 3.5

Output:
Area = 41.34 m²
Total Cost = $144.69


## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Run the Python file
4. Enter:
   - Room length
   - Room width
   - Price per square meter
5. The program will display the room area and total cost.
